# Handy-Log

# Usage
1. Install .
```javascript
npm install handy-log
OR
yarn add handy-log
```

2. Use it.
```javascript
const hl = require('handy-log')
hl.rainbow('Logs it as a rainbow')
hl.radar('Logs it as a radar')
hl.success('Recommeded for success logs')
hl.error('Recommended for error logs')
```
